## A boilerplate for react-redux application

#### Folder Structrue

```
...
src
├─ actions
├─ components
  └─ App.js
├─ containers
└─ reducers
  └─ index.js
...
```

#### Formatting Code

- `eslint` with airbnb, react-app and prettier-recommeded extension
- `prettier` format code
- `husky` make it easy to use githooks
- `pretty-quick` run prettier on changed files when commit
